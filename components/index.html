Baker Cache

The BakerCache component aims to save and load raw baker outputs as fast as possible. The most common potential use would be for an undo/redo system.

Setup

  • Set an export path where data can be saved.
  • Populate the bakers list.
// Save the current state of all bakers output textures.
Save(cacheName);

// Load a cache to all bakers.
Load(cacheName);

// Delete a cache.
Delete(cacheName);
// or
DeletAll();

// Temporary textures are kept around to speed up cacheing lots of files.
// Call this after Save/Load, or when you wont be calling Save/Load any more.
ClearTempTextures();
