Texture Smith Docs

Texture Smith is an evolving texture creation program built for Unity that can create multiple textures (i.e PBR) at once, fast enought for runtime.

Features

  • Fast enough for real time use. (Depending on texture size + count.)
  • Multiple intuitive painting techiques like projection or uv.
  • Material agnostic. Can create textures for whatever you need.
  • Heightblending. (Optional.)
  • Built in ProceduralMaterial support.
  • Built in TextMesh Pro support.
  • Optimization options, to get things running as fast as possible.
  • Use as many brushes as you want.

Potential Uses

  • Character creation systems.
  • Painting programs/art asset development.
  • Procedural texture generation.
  • Blood, dirt, explosion effects.

Caution

The techniques wont work as well if your mesh shares uvs. For instance every face of the built in Unity cube has the same uvs, so you would only be able to paint to one face.
